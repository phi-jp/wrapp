# wrapp
wrapp


## install

```
$ npm install -g cordova
$ npm install -g cordova-icon
$ npm install -g cordova-splash
$ npm install -g ios-sim
```

## setup

```
$ cordova platform add ios
```

## build

```
$ cordova build ios
```

## emulate

```
$ cordova emulate ios --target='iPhone-5s'
```
